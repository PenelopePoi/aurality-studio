# Aurality Studio

All-in-one creative super-app built as a progressive web application. Aurality combines audio production, visual design, writing, and creative tools into a single platform — replacing 30+ standalone apps.

A project of **Aurality Inc.**, a 501(c)(3) nonprofit organization.

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Progressive Web App (PWA) with offline support
- Service Worker caching

## Getting Started

Open `index.html` in a browser or serve with any static file server:

```bash
npx serve .
```
